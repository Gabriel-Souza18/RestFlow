
RestoFlow é um sistema completo de gestão para restaurantes, desenvolvido para otimizar o fluxo de trabalho desde o atendimento ao cliente até a gestão administrativa.
# Funcionalidades
## Módulo de Atendimento

* Abertura de comandas

* Gestão de mesas (Livre/Ocupada)

* Pedidos em tempo real

## Módulo da Cozinha

* Visualização de pedidos

* Controle de preparação

* Atualização de status

* Priorização de pedidos

## Módulo Financeiro

* Registro de pagamentos

* Fechamento de comandas

## Módulo Administrativo

* Gestão de funcionários

* Controle de acesso por tipo

* Monitoramento em tempo real
